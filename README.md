# Sample_PureLayout

## NoStoryBoard
StoryBoardを利用しない必要最低限のテンプレート。まずはこれをベースにPureLayoutを学ぶと良い
- Main.StoryBoard削除
- main.swift追加
- appDelegate.swift修正
- info.plist, assets, appDelegateを /Application配下に
- Build Setting(Use Legacy Swift Language Version: Yes）
- ディレクトリ構成
<pre>
 NoStoryBoard
  ├─ Application
  ├─ Compornents
  ├─ Frameworks
  └─ UI
</pre>
